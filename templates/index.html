02. about

A developer who likes things clean, fast, and working.

I'm Kimberly — a Frontend Developer and Python Developer who enjoys turning ideas into interfaces people actually want to use, and turning repetitive work into scripts that run themselves. My frontend work centers on building responsive, accessible web applications with React, Next.js, and modern CSS — the kind of interfaces that feel considered on a phone and on a 27-inch monitor alike.

On the backend, I write Python to build APIs with Flask and FastAPI, automate tedious workflows, and wrangle data with tools like Pandas, Selenium, and BeautifulSoup. I like the discipline of clean, maintainable code — code that a teammate (or future me) can read without a decoder ring — and I bring the same care to problem solving whether the bug is in a CSS grid or a broken API response.

I'm a continuous learner by default — I'm currently deepening my skills in AI and machine learning, with the goal of building intelligent automation that goes beyond simple scripts. Frontend, backend, or somewhere in between: I care about shipping software that works, reads well, and respects the person using it.
